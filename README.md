# Digit-Drop
A **Wordle-style 4-digit number guessing game** built as a PWA. Guess the secret code in 6 tries — tiles flip to show if each digit is correct, present, or absent. Features 5 swappable color themes (Void, Milk, Matrix, Cherry, Ocean), confetti on wins, animated backgrounds, and a stat tracker. Zero dependencies — pure HTML/CSS/JS.
